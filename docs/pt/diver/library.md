## Objetivo

A **Biblioteca** reune os seus materiais de aprendizagem (por exemplo, livros do curso), dependendo do que estiver disponivel para o seu perfil.

## Onde encontrar

Menu: **Diver -> Minha Biblioteca**

## Passos tipicos

1. Abra **Minha Biblioteca**.
2. Selecione um item (curso/livro).
3. Abra-o ou faca download (dependendo das acoes disponiveis).

## Problemas comuns

- Nenhum conteudo visivel: talvez ainda nao tenha itens associados ao seu perfil.
- Redirecionado para o login: sessao expirada.
