# Мой профиль: обзор

Раздел **Мой профиль** доступен из любой области (Diver/Professional/Dive Center/Distributor) через меню фото профиля.

## Где найти

В правом верхнем углу: нажмите **фото профиля** -> **Мой профиль** (и связанные пункты).

## Основные страницы

- Обзор (эта страница)
- Мой профиль
- Мои задачи (assignments)
- Моя медицинская история
- Моя безопасность
- Моя конфиденциальность

## Примечания

- Некоторые пункты могут отображаться только если они включены для вашего аккаунта (например **Мои задачи**).

<details>
<summary>Для поддержки (технические детали)</summary>

Страницы профиля используют префикс языка и фиксированный путь:

```text
GET https://user.diveraid.com/ru/user/profile
GET https://user.diveraid.com/ru/user/profile/assignments
GET https://user.diveraid.com/ru/user/profile/medical
GET https://user.diveraid.com/ru/user/profile/security
GET https://user.diveraid.com/ru/user/profile/privacy
```

Выход:

```text
POST https://user.diveraid.com/ru/logout
```

</details>
