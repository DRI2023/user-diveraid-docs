# Мои задачи

## Назначение

Назначения и приглашения, связанные с вашим аккаунтом (если включено).

## Где найти

Меню фото профиля -> **Мои задачи**

![My Tasks page](../../assets/screenshots/profile/en/my-tasks.png)

## Типичные действия

- Просмотреть ожидающие приглашения и активные сотрудничества.
- Подтвердить или отклонить приглашения.
