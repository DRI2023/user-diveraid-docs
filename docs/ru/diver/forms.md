# Дайвер: формы рейдов

## Где найти

Меню: **Дайвер -> Формы рейдов**

## Список форм

На этой странице перечислены доступные формы.

![Формы](../../assets/screenshots/diver/en/forms.png)

Типовые шаги:

1. Откройте список форм.
2. Выберите форму, чтобы посмотреть или заполнить ее (в зависимости от типа формы).

## Частые проблемы

- Форма недоступна: она может быть ограничена курсом или конкретной ролью.
