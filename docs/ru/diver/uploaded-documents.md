# Дайвер: загруженные документы

## Где найти

Меню: **Дайвер -> Загруженные документы**

## Загруженные документы

Здесь вы найдете документы, связанные с вашим профилем (обычно загруженные/зарегистрированные под вашим аккаунтом).

![Загруженные документы](../../assets/screenshots/diver/en/documents-uploaded.png)

Типовые шаги:

1. Откройте список.
2. Используйте фильтр или поиск (если доступно).
3. Откройте документ, чтобы посмотреть детали.

## Как загрузить документ

Типовые шаги:

1. Нажмите **Add a document**.
2. В **Document type** выберите тип (см. ниже).
3. Загрузите файл (PDF/JPG/PNG).
4. Заполните обязательные поля (в зависимости от типа).
5. Сохраните и проверьте документ в списке.

## Какой тип документа выбрать?

- **Общий документ (General document):** любой общий файл (разное/вспомогательный документ).
- **Сертификации для crossover (License):** сертификаты, полезные для crossover.
- **Медицинская справка дайвера (Diver medical certificate):** медицинская справка для дайвера.
- **Медицинская справка профессионала (Divemaster/Instructor):** медицинская справка для профессиональных дайверов.
- **Профессиональная страховка (Professional insurance):** подтверждение профессиональной страховки.

## Частые проблемы

- Документ не виден: возможно он еще не привязан к вашему профилю или у вас нет доступа.
- Загрузка не начинается: обновите страницу или проверьте настройки загрузок/безопасности браузера.
