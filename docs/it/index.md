# Guida Utente DiverAid

Questa guida documenta l'uso di `user.diveraid.com`.

Per scelta progettuale, la guida vive in un repository separato (`user-diveraid-docs`) e non modifica il progetto Laravel.

## Lingue

Puoi cambiare lingua dal selettore in alto (IT/EN).

## Come sono fatte le URL

La maggior parte delle pagine "backend" e' sotto un prefisso locale `/{locale}` (es. `/it/...`, `/en/...`).

Esempi:

```text
/it/diver/dashboard
/en/diver/dashboard
```

## Stato

Si parte dall'area **Diver**. Dopo tua conferma, estenderemo la guida a:

- Professional
- Dive Center
- Distributor

Vai a: [Diver > Panoramica](diver/overview.md)
