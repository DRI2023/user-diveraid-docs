# Diver: libreria

## Screenshot
![La mia libreria](../../assets/screenshots/diver/en/library.png)

## Scopo

La **Libreria** raccoglie i tuoi materiali di apprendimento (ad esempio libri/corsi), in base a cio' che e' disponibile per il tuo profilo.

## Dove si trova

Menu: **Diver -> La mia libreria**

## Passi tipici

1. Apri **La mia libreria**.
2. Seleziona un elemento (corso/libro).
3. Aprilo o scaricalo (in base alle azioni disponibili).

## Problemi comuni

- Nessun contenuto visibile: potresti non avere ancora elementi associati al tuo profilo.
- Reindirizzato al login: sessione scaduta.
