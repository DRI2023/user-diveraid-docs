# Dalgıç: Belgelerim

## Nerede bulurum?

Menü: **Dalgıç -> Belgelerim**

## Personal documents (kişisel belgeler)

Size sağlanan belgeleri burada bulabilirsiniz.

![Personal documents](../../assets/screenshots/diver/en/documents-personal.png)

Tipik adımlar:

1. Listeyi açın.
2. Bir belge seçin.
3. (Arayüze bağlı olarak) görüntüleyin veya indirin.

Yukledigini belgeler icin [Yüklenen belgeler](uploaded-documents.md) sayfasini acin.

## Yaygın sorunlar

- Belge görünmüyor: henüz profilinizle ilişkilendirilmemiş olabilir veya erişiminiz olmayabilir.
- İndirme başlamıyor: sayfayı yenileyin veya tarayıcının indirme/güvenlik ayarlarını kontrol edin.
