# Diver: mes documents

## Ou le trouver

Menu : **Mes documents**

## Documents personnels

Ici tu trouves les documents fournis pour toi.

![Personal documents](../../assets/screenshots/diver/en/documents-personal.png)

Etapes typiques :

1. Ouvrir la liste.
2. Selectionner un document.
3. Afficher ou telecharger (selon l'interface).

Pour les documents que tu as televerses, va dans [Documents telecharges](uploaded-documents.md).

## Problemes frequents

- Document non visible : pas encore associe a ton profil ou acces non autorise.
- Telechargement ne demarre pas : recharge la page ou verifie les reglages du navigateur.
