# Diver: documents telecharges

## Ou le trouver

Menu : **Documents telecharges**

## Documents televerses

Ici tu trouves les documents associes a ton profil (generalement televerses/enregistres sous ton compte).

![Uploaded documents](../../assets/screenshots/diver/en/documents-uploaded.png)

Etapes typiques :

1. Ouvrir la liste.
2. Filtrer ou rechercher (si disponible).
3. Ouvrir un document pour voir les details.

## Comment televerser un document

Etapes typiques :

1. Clique **Add a document**.
2. Dans **Document type**, choisis le type (voir ci-dessous).
3. Televerse le fichier (PDF/JPG/PNG).
4. Renseigne les champs requis (selon le type).
5. Enregistre puis verifie le document dans la liste.

## Quel type de document choisir ?

- **Document general:** n'importe quel fichier general (divers/justificatif).
- **Licence (crossover):** certifications utiles pour un crossover.
- **Certificat medical Diver:** certificat medical pour le diver.
- **Certificat medical professionnel (Divemaster/Instructor):** certificat medical pour les professionnels.
- **Assurance professionnelle:** preuve d'assurance professionnelle.

## Problemes frequents

- Document non visible : pas encore associe a ton profil ou acces non autorise.
- Telechargement ne demarre pas : recharge la page ou verifie les reglages du navigateur.
