## Objectif

La **Bibliotheque** regroupe vos supports d'apprentissage (par exemple des livres de cours), selon ce qui est disponible pour votre profil.

## Ou la trouver

Menu: **Diver -> Ma bibliotheque**

## Etapes typiques

1. Ouvrez **Ma bibliotheque**.
2. Selectionnez un element (cours/livre).
3. Ouvrez-le ou telechargez-le (selon les actions disponibles).

## Problemes courants

- Aucun contenu visible: il se peut que vous n'ayez pas encore d'elements associes.
- Redirige vers la page de connexion: session expiree.
