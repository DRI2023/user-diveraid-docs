## Proposito

La **Biblioteca** reune tus materiales de aprendizaje (por ejemplo libros del curso), segun lo que este disponible para tu perfil.

## Donde encontrarla

Menu: **Diver -> Mi biblioteca**

## Pasos habituales

1. Abre **Mi biblioteca**.
2. Selecciona un elemento (curso/libro).
3. Abrelo o descargalo (segun las acciones disponibles).

## Problemas comunes

- No hay contenido visible: puede que aun no tengas elementos asociados.
- Redirigido al inicio de sesion: sesion caducada.
