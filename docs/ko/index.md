# DiveRaid 사용자 가이드에 오신 것을 환영합니다

이 가이드는 `user.diveraid.com`에서 문서, 코스, 인증, 다이빙 로그, 구매를 관리하는 방법을 안내합니다.

## 주요 섹션

- [대시보드](diver/dashboard.md)
- [내 코스](diver/courses.md)
- [내 인증](diver/certifications.md)
- [내 라이브러리](diver/awards.md)
- [내 다이빙 로그](diver/dive-logs.md)
- [내 문서 / Uploaded documents](diver/documents.md)
- [무료 학습 상점](diver/free-learnings.md)
- [스토어](diver/store.md)
- [레이드 양식](diver/forms.md)

## 시작하기

1. **포털 접속:** **다이버 -> 대시보드**로 이동해 계정이 차단되지 않았는지 확인합니다 (예: 이메일 미인증).
2. **프로필 설정:** **내 문서 / Uploaded documents**에서 필요한 파일을 업로드합니다.
3. **인증 확인:** **내 인증**에서 카드 또는 이력을 확인합니다.
4. **다이빙 로그 추가:** **내 다이빙 로그**에서 새 로그를 만든 뒤 확인/수정합니다.

## 언어

상단의 언어 선택기에서 변경할 수 있습니다 (EN/IT/DE/FR/ES/NL/ZH/KO/TR/PT/PL/RU).


## 상태

현재는 **다이버** 영역부터 시작합니다. 검토 후 다음 영역으로 확장합니다:

- Professional
- Dive Center
- Distributor
