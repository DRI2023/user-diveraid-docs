# 다이버: 레이드 양식

## 어디에서 찾을 수 있나요

메뉴: **다이버 -> 레이드 양식**

## 양식 목록

이 페이지에는 사용 가능한 양식(Form)이 목록으로 표시됩니다.

![Forms](../../assets/screenshots/diver/en/forms.png)

일반적인 단계:

1. 양식 목록을 엽니다.
2. 양식을 선택해 내용을 확인하거나(양식 유형에 따라) 작성합니다.

## 자주 발생하는 문제

- 양식을 사용할 수 없음: 특정 코스 또는 특정 역할로 제한되어 있을 수 있습니다.
