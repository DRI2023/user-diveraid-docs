# 潜水员：已上传的文件

## 在哪里找到

菜单：**潜水员 -> 已上传的文件**

## 已上传的文件

这里通常包含与你的账号资料关联的文件（一般是你上传/登记到个人资料下的文件）。

![已上传的文件](../../assets/screenshots/diver/en/documents-uploaded.png)

典型操作：

1. 打开列表。
2. 使用筛选/搜索（如果有）。
3. 打开文件查看详情。

## 上传文件

典型操作：

1. 点击 **Add a document**。
2. 在 **Document type** 中选择类型（见下方说明）。
3. 上传文件（PDF/JPG/PNG）。
4. 按类型填写必填信息。
5. 保存后，在列表中检查该文件是否出现。

## 应该选择哪种文件类型？

- **通用文件：** 任意通用文件（杂项/辅助材料）。
- **证书（crossover）：** 用于 crossover 的相关认证/证书。
- **潜水员体检证明：** 潜水员用体检证明/医疗证明。
- **专业人士体检证明（潜水长/教练）：** 专业潜水员用体检证明/医疗证明。
- **专业保险：** 专业保险证明文件。

## 常见问题

- 看不到文件：文件可能还未关联到你的资料，或你没有权限。
- 无法下载：刷新页面，或检查浏览器下载/安全设置。
