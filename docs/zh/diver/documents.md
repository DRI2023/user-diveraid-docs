# 潜水员：文件

## 在哪里找到

菜单：**潜水员 -> 常用文件**

## 常用文件（个人文件）

这里通常包含系统提供给你的文件（例如课程相关资料等）。

![常用文件](../../assets/screenshots/diver/en/documents-personal.png)

典型操作：

1. 打开列表。
2. 选择一个文件。
3. 查看或下载（取决于界面功能）。

你上传的文件请查看：[已上传的文件](uploaded-documents.md)。

## 常见问题

- 看不到文件：文件可能还未关联到你的资料，或你没有权限。
- 无法下载：刷新页面，或检查浏览器下载/安全设置。
