# 欢迎使用 DiveRaid 用户指南

本指南帮助你使用 `user.diveraid.com` 来管理文件、课程、认证、潜水日志和购买内容。

- [线上商店](diver/store.md)
- [表格工具](diver/forms.md)

## 从哪里开始

1. **登录门户：** 打开 **潜水员 -> 汇总信息**，确认账号未被限制（例如邮箱未验证）。
2. **完善资料：** 在 **常用文件 / 已上传的文件** 中上传或查看你的文件。
3. **查看认证：** 进入 **我已获得的潜水员级认证**，查看卡片和历史记录。
4. **填写潜水日志：** 进入 **我的潜水日志**，新建日志并保存，然后可随时回看/编辑。
